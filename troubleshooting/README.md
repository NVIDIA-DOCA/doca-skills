# Troubleshooting Topics

Applies to: `NVIDIA-DOCA/doca-skills/troubleshooting`
Read when: choosing a troubleshooting topic guide
Load next: `getting-started/troubleshooting.md`

Use this folder when a build, dependency, or validation step fails.

| Topic | Use When |
| --- | --- |
| `build-issues.md` | Starting from an unknown build failure. |
| `build-validation.md` | Separating discovery, configure, compile, and runtime evidence. |
| `meson-build-issues.md` | Debugging Meson dependency, source staging, or target-selection failures. |

Keep reports concrete: exact command, focus path, failing dependency or source file, and the next safe command.

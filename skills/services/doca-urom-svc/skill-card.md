## Description: <br>
Use when operating the DOCA UROM Service container on a BlueField Arm to run remote memory ops (puts, gets, atomics, collectives) that paired hosts enqueue via the host-side doca-urom library — pulling the NGC container, choosing UCX-component surface, sizing the enqueue queue, wiring Comch endpoint pairing with doca-urom, or pinning host-library + service versions per the DOCA Compatibility Policy. <br>

This skill is ready for commercial/non-commercial use. <br>

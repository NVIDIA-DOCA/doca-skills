## Description: <br>
Guides agents through hands-on DOCA DPA host-side work on a BlueField — creating the doca_dpa Core context, loading DPACC-compiled DPA application images, creating DPA threads, launching kernels, draining completions, running capability discovery, and debugging DOCA_ERROR codes from doca_dpa_* calls. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
External developers building applications that consume the DOCA DPA library from the host side — using doca_dpa_* to load DPA application images, launch DPA kernels, and exchange data with the DPA processor on a BlueField. <br>

## How to re-audit

URL HEAD validity over every URL in every skill file (including
`SKILL.md` in this folder) is enforced on every commit by NVIDIA's
internal release CI in `--check-urls` mode whenever outbound network
is available; the audit log row above is bumped on a DOCA release
after the live `docs.nvidia.com/doca/sdk/index.html` redirect target
has been re-confirmed and any new umbrella pages added.

The script also enforces the *public-sources only* contract from
[`AGENTS.md`](../../AGENTS.md) ground rule #1 by allowlisting NVIDIA
hosts and rejecting URLs / paths that mention internal tooling
vocabulary; that part runs without network.

When you record a new audit row in the table above, also bump the
visible *Last audit* date in [`SKILL.md`](SKILL.md) `## URL audit` so
runtime agents can see how stale the routing is without reading this
file.
